# Tienda Retro

## Descripción breve
Tienda Retro es una aplicación estilo tienda desarrollada como trabajo práctico de programación. Permite recorrer un catálogo de productos, ver detalles, agregar al carrito, aplicar filtros y finalizar la compra mediante un formulario de contacto.

## Tecnologías utilizadas
- React
- React Router DOM
- React Bootstrap
- JavaScript
- Vite

## Instrucciones para instalar y correr el proyecto
1. Clonar el repositorio:
    git clone https://github.com/LucaAR24/tienda-retro.git

2. Entrar a la carpeta del proyecto:
    cd tienda-retro

3. Instalar dependencias:
    npm install

4. Ejecutarlo en modo desarrollo:
    npm run dev

5. Abrir en el navegador:
http://localhost:5173
